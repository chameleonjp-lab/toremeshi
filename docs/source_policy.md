# トレメシ 根拠と判定の管理ルール v1.0

## 1. 目的

この文書は、トレメシの食品・プロテイン・正解条件を、あとから見直せる形で管理するためのルールです。

プロのトレーナー試験に使う前提なので、感覚的な「よさそう」では正解にしません。
数値、条件、根拠の3つがそろったものだけを正解にします。

## 2. 根拠の優先順位

1. 文部科学省 食品成分データベース
2. 日本食品標準成分表（八訂）増補2023年
3. 消費者庁の食物アレルギー表示情報
4. スポーツ栄養に関する査読論文・国際的な声明
5. 実在製品ではない標準モデル値

加工食品、外食風セット、プロテインは製品差が大きいため、実在商品の一般化はしません。
問題文に出す標準モデル値を正本として扱います。

## 3. 判定の優先順位

判定は必ず次の順で行います。

1. 安全除外条件
2. 問題タイプの主条件
3. 1食分の栄養素数値
4. タグ
5. 解説可能性

たんぱく質が多くても、乳を避ける相談者に乳由来プロテインを正解にしません。
夜遅い問題では、カフェイン入り飲料を正解にしません。

## 4. 安全除外タグ

最低限、次のタグを使います。

- contains_milk
- contains_egg
- contains_wheat
- contains_buckwheat
- contains_shrimp
- contains_crab
- contains_peanut
- contains_walnut
- contains_cashew
- contains_tree_nuts
- contains_soy
- contains_fish
- contains_shellfish

`contains_tree_nuts` は、くるみ・カシューナッツ以外の木の実をまとめて扱う補助タグです。

## 5. 公開前チェック

試験問題として使う前に、各食品で次を確認します。

- 1食分の量が明確か
- 栄養素数値の出典が追えるか
- 加工食品・セット品の計算式が残っているか
- アレルゲンタグが入っているか
- F01〜F09のどの条件で正解になるか説明できるか

`nutrition_value_status` が `final_verified` ではないものは、公開試験の正解根拠に使いません。
